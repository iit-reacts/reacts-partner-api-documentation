  2. Interface de session – Zone de travail principale

 La zone de travail principale peut être divisée en plusieurs parties:

Panneau d’éléments (voir le document “ Panneau d’éléments “ pour plus d’information)

Panneau d’actions (voir le document “ Panneau d’éléments “ pour plus d’information) 

Fenêtre de la scène

 Activer ou désactiver les haut-parleurs et le microphone, vignette de la caméra, accéder au bureau/à l’application partagée, clavardage, barre d’état

Les options suivantes s’affichent sur la barre de menu

 **Saisir un instantané ou enregistrer une courte vidéo :**

Vous permet de prendre une photo de n’importe quel élément sur une toile. Tous les instantanés saisis au cours d’une session s’affichent dans la galerie d’instantanés et sont stockés dans la bibliothèque d’instantanés sous un dossier portant la date à laquelle l’instantané a été saisi.

Vous pouvez consulter le tutoriel vidéo Instantanés (photo: https://www.youtube.com/watch?v=vh8oougq8t8, video : https://www.youtube.com/watch?v=NupIc5ektvw)

Gestion des couches:

Si vous avez plusieurs éléments sur une même scène, le menu Couches vous aidera à gérer ces éléments. 

Verrouiller ou déverrouiller votre arrière-plan:

 Vous pouvez insérer un objet ou une entrée sur une toile en tant qu’arrière-plan ou en tant qu’élément.

Pour passer facilement de la gestion de l’Arrière-plan à la gestion des Éléments, vous pouvez verrouiller/déverrouiller votre arrière-plan.

 Vous pouvez consulter le tutoriel vidéo Arrière-plan https://www.youtube.com/watch?v=0GzVNPuSqPo

Modifier le rapport de dimensions de la caméra:

Par défaut, le rapport est établi à la valeur la plus appropriée, mais vous pouvez le modifier à partir de la barre de menu.

 Veuillez noter que le rapport doit être modifié avant l’ajout d’éléments. Autrement, les éléments en place seront supprimés.

Vous pouvez consulter le tutoriel vidéo Ajuster le format d’une fenêtre de scène (ratios recommandés: 4:3, 16:9) (https://www.youtube.com/watch?v=Gw81GgfeYV8)

Ajuster la couleur et la taille du pointeur:

Vous pouvez consulter le tutoriel vidéo Pointeur (https://www.youtube.com/watch?v=-xbKieQDV_c)
