L’une des fonctionnalités uniques de Reacts vous permet d’intégrer des objets 3D dans vos présentations.

Pour ajouter des objets 3D dans vos sélections éclair, accédez au panneau d’éléments, sélectionnez l’option Objets 3D et cliquez sur le bouton « + ». Des modèles d’objets 3D sont fournis avec la plateforme :

Remarque : Vous ne pouvez pas insérer un objet 3D en tant qu’arrière-plan.

Vous pouvez également importer vos propres objets 3D dans Reacts – consultez les directives dans le document “Directives pour l’importation de vos objets 3D dans Reacts”.

Lorsque vous aurez ajouté un objet 3D à une scène, plusieurs options vous seront proposées pour les gérer et les manipuler :

  • Pour faire pivoter un objet 3D, utilisez votre souris en maintenant le bouton Ctrl de votre clavier enfoncé;
  • Faites un clic droit sur un objet avec votre souris pour afficher les options suivantes : Renommer, Arranger, Redimensionner, Faire pivoter, Modifier l’opacité, Agrandir, Réinitialiser et Supprimer:
