Le panneau d’actions proposera différentes options si vous créez/modifiez une session, si vous essayez d’établir une connexion avec un contact ou prenez part à une session en direct.

 Ajouter une nouvelle scène à votre présentation

Vous pouvez consulter le tutoriel vidéo Ajouter une nouvelle scène (https://www.youtube.com/watch?v=-C8T2pEwhn8&t=1s)

Établir une connexion avec un autre utilisateur Reacts à partir de votre liste de contacts

Après avoir sélectionné l’option Ajouter à l’appel, vous accèderez à une liste de vos contacts. Il vous suffit de choisir l’un de vos contacts dans la liste et d’établir une connexion.

Vous pouvez consulter le tutoriel vidéo “Établir une connexion avec un contact” (à partir de l’interface principale de la session) (https://www.youtube.com/watch?v=KjuCd2G0HGI)

Gérer des sessions existantes ou créer de nouvelles sessions

Vous pouvez consulter le tutoriel vidéo “Session” (Nouveau / Ouvrir / Enregistrer / Enregistrer sous / Fermer) des sessions (version anglaise) (https://www.youtube.com/watch?v=QOxIu46RltU)

Ajuster les paramètres audio/vidéo, récupérer des entrées supprimées

Vous pouvez consulter le tutoriel vidéo “Comment ajuster les paramètres de l’interface principale (https://www.youtube.com/watch?v=lwsSFZA7ges) 

Commencer une session au cours d’un appel

 Une fenêtre de scène s’ouvrira pour vous permettre de créer votre session

Passer au mode Face à face en mode Session

 En mode Face à face, une entrée de caméra s’affichera à l’écran pour vous permettre de discuter avec votre contact.

Vous pouvez consulter le tutoriel vidéo “Face-à-Face” (https://www.youtube.com/watch?v=QbKl31obk0I)
