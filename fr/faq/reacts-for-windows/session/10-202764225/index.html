Démarrer une nouvelle session ou ouvrir une session existante 

Nouvelle session – Ouvre la zone de travail principale vous permettant de créer une session ou d’établir une connexion avec l’un de vous contacts.

Sessions sauvegardées – Vous permet de consulter, de gérer ou de modifier des sessions sauvegardées. Faites un clic droit sur une session pour l’ouvrir, la supprimer ou la renommer.

Session planifièes - Vous permet d’organiser une session (à titre d’hôte) avec l’un ou plusieurs de vos contacts Reacts ou de prendre part (en tant que participant) à une session à laquelle on vous a invité.

Pour en savoir davantage sur la planification de sessions, visionner : Comment planifier un évènement (https://youtu.be/GYNZ0bttKHA)
