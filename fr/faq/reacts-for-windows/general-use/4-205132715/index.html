Hauts parleurs portables (optionnel) (pour une qualité audio supérieure à celle des haut-parleurs intégrés. Une variété de modèle est disponible sur le marché.)

Convertisseur de signal vidéo numérique (DVI/VHA /HDMI) en signal USB (pour les appareils d’échographie, les appareils à ultrasons, les moniteurs, les bronchoscopes, etc.). Le signal USB est automatiquement détecté par Reacts comme une entrée vidéo (n’oubliez pas d’installer le pilote du convertisseur) : Épiphan AV.iO HD

Vous pouvez vous référer à cette vidéo en anglais (https://www.youtube.com/watch?v=aePGHV76M6I) expliquant comment utiliser un convertisseur de signal vidéo avec Reacts.

** D’autres convertisseurs du même type sont offerts sur le marché

Convertisseur de signal vidéo analogique en signal USB. Le signal USB est automatiquement détecté par Reacts comme une entrée vidéo (n’oubliez pas d’installer le pilote du convertisseur). Plusieurs modèles sont offerts sur le marché et peuvent être trouvés sur internet.
