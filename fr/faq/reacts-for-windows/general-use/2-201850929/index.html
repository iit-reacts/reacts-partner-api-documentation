Si vous voulez trouver quelqu’un dans votre liste de contacts Reacts, saisissez le nom ou l’adresse courriel de l’utilisateur dans le champ de recherche :

  • Vous pourrez ainsi filtrer les utilisateurs de votre liste de contacts;

**Si vous voulez ajouter un contact:
**

Cliquer sur l’icone d’ajout de contact:

Suivre les instruction indiquées à l’écran :

L’utilisateur recevra un avis l’informant de la nouvelle demande de contact:

L’utilisateur peut accepter ou ignorer la nouvelle demande de contact:

Si vous voulez inviter quelqu’un à utiliser Reacts, choisissez l’option « Inviter à utiliser Reacts»:

Le nouvel utilisateur recevra un courriel contenant des directives pour s’inscrire à Reacts et télécharger l’application.

Si vous souhaitez supprimer un contact existant, sélectionnez l’option « Modifier», cliqué sur la croix correspondant au contact à supprimer puis confirmez votre choix :
