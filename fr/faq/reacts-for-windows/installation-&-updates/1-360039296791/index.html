Prenez note que l’application windows est munie d’une mise à jour automatique, donc il est fort probable que les versions soient téléchargées et installées sans avoir besoin d’action. 

Dans l’éventualité où la version que vous avez ouverte n’est pas la plus récente, voici les étapes à suivre: 

  1. Se connecter à Reacts

  2. une fois dans l’application Reacts Windows: une nouvelle fenêtre “Une nouvelle version de Reacts est nécessaire” s’affichera.

  3. Cliquer sur le bouton “Quitter Reacts” de la fenêtre :

  4. Attendre 5 min.

  5. Réouvrir Reacts.
  6. La mise à jour est complétée et vous pouvez vous reconnecter.
