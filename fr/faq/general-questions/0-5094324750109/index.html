Reacts est disponible sur :

- Google Chrome (Mac et Windows) : une installation de Reacts n’est pas requise

- Sur appareils mobiles (appareil mobiles iOS et Android) : une installation Reacts est requise. L’application Reacts est disponible sur l’AppStore et le Google PlayStore. Vous pouvez utiliser les liens présents dans le tableau si dessous pour trouver Reacts sur l’AppStore ou le Google PlayStore. Vous pouvez aussi écrire “Reacts” dans la barre de recherche de votre application AppStore ou Google PlayStore pour trouver l’application.

- Sur Windows: une installation Reacts est requise. Vous pouvez utiliser le lien présent dans le tableau si dessous pour télécharger Reacts pour Windows.

Avant d’installer Reacts, veuillez consulter la page “Exigences du système” pour en savoir plus sur les exigences sytèmes.

Pour en savoir plus sur les fonctionnalités disponibles sur chacunes de nos plateformes, veuillez consulter la page “Plateformes et fonctionnalités”.
