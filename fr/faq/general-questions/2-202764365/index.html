  2. Comment modifier les paramètres et les renseignements de compte dans Reacts?

Cet article est également disponible dans les langues suivantes:

  • English
  • Français

Modifiez vos paramètres de compte et votre langue d’utilisation, configurez votre microphone, votre caméra et vos haut-parleurs/votre sortie audio, récupérez des entrées supprimées ou modifiez vos paramètres de réseau.

Voici les étapes pour accéder à vos paramètres : 

Windows platform

Reacts WebApp

Reacts Mobiles

Comment modifier vos paramètres de compte?

Sélectionnez l’option Mon compte du menu de configuration pour:

Windows platform

Reacts WebApp

Reacts Mobile

 Modifier – cliquez sur le bouton Modifier si vous souhaitez modifier vos renseignements personnels.

Veuillez noter que vous ne pouvez pas modifier votre adresse courriel. Communiquez avec le soutien Reacts si vous devez la modifier.

 Changer d’avatar – choisissez cette option pour choisir un autre avatar

 Choisir l’information que vous souhaitez afficher en ligne au moyen de l’option « Basculer »

 Modifier votre mot de passe

Vous pouvez consulter le tutoriel vidéo Mon compte (version anglaise) (https://www.youtube.com/watch?v=X0X96Yqc0FE&t=1s)

Comment modifier vos paramètres de langue?

Lorsque vous ouvrez une session Reacts pour la première fois, votre langue par défaut sera la même que celle que vous avez choisie lors de votre inscription. Pour changer votre langue par défaut, sélectionnez l’option Paramètres de langue, choisissez la langue de votre choix et confirmez votre sélection.

Reacts Windows

Reacts WebApp

Note : Reacts mobile utilisera la même langue que votre appareil. Si la langue de votre appareil n’est pas disponible dans Reacts, Reacts sera en anglais.

Comment configurer ou ajuster vos entrées/sorties audio/vidéo?

Pour changer vos paramètres Audio et Vidéos, veuillez sélectionner l’option Audio / Vidéo.

Reacts Windows

Reacts WebApp

 Choisissez quel appareil vous souhaitez configurer :

 Microphone

 Haut-parleurs/Sortie

 Caméra – choisissez quelle caméra vous utiliserez pour le mode Face à face. 

Vous pouvez également consulter le document “Comment ajuster les paramètres audio/vidéo?”

Pour des paramètres de caméra supplémentaires, consultez le document : “Paramètres vidéo / propriétés de la caméra”

Vous pouvez consulter le tutoriel vidéo Paramètres audio-vidéo (version anglaise)(https://www.youtube.com/watch?v=HQA9GSfLeuM)

Comment modifier vos paramètres de réseau?

Configurez vos paramètres de réseau et de serveur au besoin (en fonction du niveau de sécurité de l’emplacement depuis lequel vous vous connectez)

Vous pouvez consulter le tutoriel vidéo Paramètres du réseau (version anglaise)(https://www.youtube.com/watch?v=4Nj9aY2InJU)

Note: Reacts WebApp utilisera les mêmes paramètres que ceux présents dans les paramètres de Chrome.

Reacts mobile utilisera les mêmes paramètres que ceux présents dans les paramètres de votre téléphones.

Comment récupérer vos entrées supprimées?

Sélectionnez l’entrée dont vous avez besoin dans la liste déroulante et sélectionnez l’option Réactiver.

Vous pouvez consulter le tutoriel vidéo Entrées supprimées (version anglaise)(https://www.youtube.com/watch?v=LkHJjtOrZqg)

Note: Les fonctionnalités de suppression et reactivation d’entrées sont uniquement disponibles sur Reacts Windows.
