APERÇU

Ce tutoriel vous expliquera comment ajouter plusieurs contacts en même temps à vos machines à l’aide d’un fichier CSV.

PRÉREQUIS

  • Vous devez dans un premier temps accéder au tableau de bord Reacts.
  • Avoir une machine dans votre institution
  • Vous devrez avoir préparer votre fichier CSV

Étapes à suivre pour préparer votre fichier CSV

  1. Créer un nouveau document Excel

  2. Entrez chaque information (Prénom, nom, courriel) dans une case différente, de gauche à droite, puis aller à la ligne suivante pour le prochain contact

  3. Appuyez sur “Fichier”

  4. Appuyez sur “Enregistrer sous”, puis “Parcourrir”

  5. Appuyez sur la flèche pour faire aparaître les choix de format et choisissez CSV UTF-8

  6. Entrez un nom pour votre fichier, puis sauvegarder

Étapes à suivre pour ajouter plusieurs contacts à vos machines à l’aide d’un fichier CSV

  1. Depuis l’onglet abonnements, choisissez l’unité d’organisation qui contient votre machine

  2. Pour afficher vos machines, appuyez sur l’onglet “Machine”

  3. Appuyez sur les 3 points verticaux au bout de la ligne de la machine pour laquelle vous souhaitez ajouter des contacts, puis sélectionnez “Gérer les contacts”

  4. Appuyez sur le bouton orange “ajouter des contacts”

  5. Appuyez sur “Import CSV”

  6. Choisir le fichier que vous avez préalablement créé, puis appuyez sur “ouvrir”

  7. Appuyez sur “appliquer”

  8. Lorsque vous verrez des crochets verts qui signifient que tout a bien fonctionné, il ne vous reste qu’à appuyer sur “Terminé”
