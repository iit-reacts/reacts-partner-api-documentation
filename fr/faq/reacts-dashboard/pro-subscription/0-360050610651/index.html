APERÇU

En tant qu’administrateur vous pouvez assigner des abonnements invités et décider lesquels de vos utilisateurs PRO pourront être en communication avec ces invités.

PRÉ-REQUIS

Vous devez dans un premier temps accéder au tableau de bord Reacts.

Vous devez de plus avoir des abonnements PRO valides (non expirés) dans votre organisation.

ASSIGNER DES ABONNEMENTS INVITÉS DANS UN GROUPE

1. Sélectionner “PRO et Invités” dans le menu de gauche

2. Sélectionner l’unité d’organisation

dans laquelle vous souhaitez que votre groupe se trouve (l’unité d’organisation dans laquelle le groupe sera ajouté est sélectionnée en noir)

3. Cliquer sur “Groupe d’invités”

4. Cliquer sur “Ajouté un groupe d’invité”

5. Suivre les étapes indiquées 

Donner un nom au groupe

Ajouter des PRO

Assigner des Invités

Cliquer sur “Suivant” pour avoir un récapitulatif puis sur “Sauvergarder” pour sauvegarder vos changements

Note importante : les pro et les guests de votre groupe pourront se voir dans Reacts À CONDITION que les invités aient acceptés votre abonnement lors de leur connexion Reacts (cette acceptation peut être vu dans votre tableau de bord, à côté de l’abonnement de l’utilisateur).
