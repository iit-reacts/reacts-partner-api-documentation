APERÇU

Le tableau de bord Reacts vous permet de gérer la structure de votre organisation en y créant de nouvelles Unités d’Organisation.

Vous trouverez dans ce document les étapes à suivre pour créer des unités d’organisation.

PRÉ-REQUIS

Vous devez dans un premier temps accéder au tableau de bord Reacts.

Veuillez consulter notre tutoriel sur comment accéder à votre tableau de bord au besoin.

ÉTAPES POUR CRÉER UNE NOUVELLE UNITÉ ORGANISATIONNELLE

1. Cliquer sur les 3 boutons horizontaux situés à côté du nom de votre organisation

2. Cliquer sur “Gestion de la structure”

3. Au besoin, vous pouvez ouvrir la structure de votre organisation en cliquant sur la flèche située à gauche de celle-ci

4. Cliquer sur le symbole “+” à côté de l’Unité d’Organisation pour laquelle vous souhaitez créer une nouvelle sous-unité.

5. Indiquer le nom de votre nouvelle Unité d’Organisation

6. Cliquer sur “Ajouter”

Votre nouvelle Organisation apparaîtra dans la liste.

Dans l’exemple ci-dessous : 

L’Unité d’Organisation “ Mon Centre hospitalier” à 3 sous Unités d’organisation :

  • Cardiologie
  • Oncologie
  • Pédiatrie
