APERÇU

Ce tutoriel vous expliquera comment ajouter des droits d’administrateurs.

PRÉREQUIS

  • Vous devez dans un premier temps accéder au tableau de bord Reacts. Vous pourrez trouver les étapes pour y accéder dans le tutoriel suivant : Comment accéder à mon tableau de bord
  • Vous devez possèder des droits d’administrateur dans l’organisation (ou unité d’organisation) où vous souhaitez ajouter un administrateur

ÉTAPES À SUIVRE POUR AJOUTER DES DROITS D’ADMINISTRATEUR

  1. Choisissez l’onglet “Permissions”

  2. Cliquez sur le bouton orange “ajouter un administrateur”

  3. Cliquez sur les petites flèches afin de voir toutes vos unités d’organisation disponibles

  4. Sélectionnez à quel niveau vous désirez ajouter les droits d’administrateur, puis cliquez sur “sauvegarder”

Vous pouvez avoir plus de détail entre les deux types de droits en cliquant sur le point d’interrogation situé à coté de “Type de droit d’administrateur”

5. Lorsque vous verrez une confirmation au haut de l’écran, vous pourrez appuyer sur “fermer”

ÉTAPES À SUIVRE POUR RETIRER DES DROITS D’ADMINISTRATEUR

  1. Choisissez l’onglet “Permissions”

  2. Cliquez sur “Ouvrir” au bout de la ligne de l’administrateur que vous souhaitez modifier

  3. Cliquez sur les petites flèches afin de voir tous vos unités d’organisation disponible

  4. Décochez toutes les cases pour retirer complètements les droits d’administrateurs, puis appuyez sur “Sauvegarder”

  5. Lorsque vous verrez une confirmation au haut de l’écran, vous pourrez appuyer sur “fermer”
