PRÉREQUIS

Vous devez dans un premier temps accéder au tableau de bord Reacts.

ÉTAPES POUR ACCÉDERS À LA FONCTIONNALITÉ D’ASSIGNATION

1. Cliquer sur l’unité d’organisation dans lequel vous voulez assigner les abonnements

2. Cliquer sur ouvrir

3. Cliquer sur « assigner des abonnements »

ASSIGNER LES ABONNEMENTS

1. Choisir qui détient l’abonnement

Vous devez choisir si votre organisation détiendra ou non le compte Reacts à qui sera assigné l’abonnement.

Option A: Vous pouvez choisir cette option si vous assignez un abonnement à une adresse courriel qui n’a pas déjà un compte Reacts. (S’il y a déjà un compte qui existe, la propriété de celui-ci est au propriétaire du compte)
Note: Si vous avez besoin d’obtenir la propriété d’un compte qui existe déjà, veuillez contacter notre équipe de support Philips

Option B: Vous pouvez choisir cette option si vous assignez un abonnement à une adresse courriel possédant déjà un compte Reacts.
Note: Si vous avez besoin d’obtenir la propriété d’un compte assigné sous l’option B, veuillez contacter notre équipe de support Philips

2. Entrer les informations requises

Choisissez le type d’abonnement, entrer l’adresse courriel, le nom et prénom de la personne à qui vous souhaitez assigner l’abonnement.

3. Cliquer sur Assigner

Note importante: Si vous souhaitez ajouter plusieurs abonnements à la fois, vous pouvez cliquer sur le bouton « + » sous la première ligne d’abonnement.

 Nous vous conseillons aussi d’utiliser notre fonctionnalité d’ajout par CSV. Veuillez consulter notre tutoriel relatif à l’utilisation d’un CSV au besoin.

Une fois l’abonnement assigné, votre utilisateur recevra un courriel l’en informant et lui demandant de finaliser la création de son compte si cela n’est pas déjà fait.
